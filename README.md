# IDMPhotoBrowser
打包IDMPhotoBrowser，使其支持Carthage

# Carthage
在Cartfile中添加下面语句
```
github "damonZDM/IDMPhotoBrowser"
```
在终端执行
```
carthage update --platform iOS
```
